1.内容
DIGAの録画番組名を纏めて変更する作業を支援するファイル群です。

(1) 番組名のダンプ出力を支援するツール
・diga-dump.js
　DIGAの番組編集画面でPF12を押して「Developer Tools」を出し、
　Consoleに本ファイル内容をペーストすることで、番組情報を
　ファイルとして出力します。(ブラウザがダウンロードする)
　詳細説明や注意点は、ファイル先頭記載のコメントを参照のこと。

(2) ダンプ出力に対する番組名変更を支援するツール
・rename-tinfo.html
　RegExpベースのルールに従いダンプ出力の書換えを行うツールのhtml。
　詳細説明や注意点は、Javascriptファイル先頭記載のコメントを参照のこと。

・rename-tinfo.js
　本ツールの処理を行うスクリプトt。

(3) DIGAに番組名変更指示を出して、実際に番組名を書換えるツール
・diga-rename.html
　メイン画面のhtmlファイル。本ファイルをChromeで開き、
　右側のフレームでDIGAにログインし、番組編集ボタンを押す。
　番組名一覧が表示されたら、左側フレームの「番組名書換え開始！」
　ボタンを押すと、順次、番組名を書き換える。(8秒/1番組 (変更可能))

・diga-rename-leftF.html
　サブ画面(左側フレーム)のhtmlファイル。

・diga-rename.js
　番組名の変更を行うためのスクリプト。

・diga-tinfo.js
　変更内容を記載するファイル。詳細は後述。
　上記(2)のツールを使って、中身を作成する。

2.diga-tinfo.jsの作成方法
上記1.(1)でダンプ出力したファイルにおいて、番組名を修正し、
それを元に、下記のように整形したものを格納願います。
手作業でもできますが、1.(2)記載のツールを使うと便利です。

【例】
・整形前：
	No.	録画日	曜日	CH	開始	番組名	番組ID
	0410	16/02/14	日	地上D?061	18:00	世界遺産	10003864
	0411	16/02/14	日	地上D?061	18:30	夢の扉＋	10003867

・整形後：
    var tinfo = [
        ["No.", "録画日", "曜日", "CH", "開始", "番組名", "番組ID"],
        ["0410", "16/02/14", "日", "地上D?061", "18:00", "世界遺産", "10003864"],
        ["0411", "16/02/14", "日", "地上D?061", "18:30", "夢の扉＋", "10003867"],
    ];
